# Admin Home Page 

This is the main dashboard landing page for administrators. Upon login, admins are redirected here to view a comprehensive overview of the portfolio platform's performance and content.

Dashboard widgets:

- Total 3D Modules: count of uploaded and published 3D modules
- Portfolio Website Visitors: total unique visitor count and trends
- Module Engagement: likes, dislikes, and view counts per module
- Recent Uploads: list of recently added 3D modules
- Contact Messages: unread message count from the Contact form
- System Status: basic health indicators (uptime, errors, etc.)

Navigation:

The sidebar provides quick navigation to key admin functions:

- [New 3D modules](./New3DmodulesSubpage.md) – upload and manage new 3D modules with sketches and metadata
- [Settings](./SettingsSubpage.md) – manage account security, profile, contact details, contact form submissions, and preferences

All admin operations maintain an audit log for security and accountability.
